### position
- static : 정적 default
- relative : static위치에서 offset만큼 이동
- absolute : static이 아닌 부모 아래위치에서 offset만큼 이동

### display
- inline : 한줄에 나란히
- block : 한줄에 하나
- inline block : 한줄에 나란히 들어가지만 wight, height 적용 가능
